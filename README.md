# 智能书签助手 Chrome 扩展

AI 智能分析网页内容，自动分类添加书签。

## 功能

- 📄 **读取当前页面** - 自动获取标题和URL
- 🔍 **AI智能分析** - 抓取网页内容，AI分析生成名称
- 📂 **智能分类** - 读取现有书签分类，推荐最佳分类
- ✨ **一键添加** - 直接添加到Chrome书签

## 安装方法

1. 打开 Chrome 浏览器
2. 地址栏输入 `chrome://extensions/`
3. 右上角打开 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `bookmark-extension` 文件夹
6. 完成！工具栏会出现扩展图标

## 使用方法

1. 打开任意网页
2. 点击工具栏的扩展图标
3. 点击 **AI 智能分析**
4. 确认或修改名称和分类
5. 点击 **添加书签**

## 技术栈

- Chrome Extension Manifest V3
- Vercel Design System 风格
- AI API 智能分析
